Buttons



Various button colors with a solid background.

Various buttons styled with a color outline.

An example you would find on a gene or genome details page. Outline buttons can be used to indicate that a button has been clicked and the action has been executed.

These buttons are used for adding or removing genes, genomes, functions or scaffolds to the cart when those items are viewed in a list. They are analogous to the buttons in the example above. Below is an example of how they look in a table of genomes.
