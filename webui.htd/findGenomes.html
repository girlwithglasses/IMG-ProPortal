Examples

- "pseudomonas" as Genome Name retrieves all genomes with the substring "pseudomonas" such as "Pseudomonas syringae B728a".
- "62977" for NCBI Taxon ID retrieves "Acinetobacter sp. AD1".
- "NC_008009, NC_008010, NZ_AAKW01000001" as Scaffold External Accession will retrieve genomes that have scaffolds with those accession ID's.
- "arch" for Domain retreives all Archaea.
- "proteo" for Phylum retrieves all proteobacteria.
- "finished" for Sequencing Status retrieves all finished genomes.
- "draft" for Sequencing Status retrieves all finished genomes.
- "JGI" for Sequencing Center retrieves all genomes sequenced at JGI.
- "2.0" for IMG Release retrieves all genomes with IMG release 2.0
